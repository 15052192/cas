OpenID Connect Protocol

You can configure the CAS server with:

  • OIDC client support, which means authentication can be delegated through a link on the
    login page to OpenID Connect provider.
  • OIDC server support, which means you will be able to communicate with your CAS server
    through the OpenID Connect protocol, having CAS act as an OP.
